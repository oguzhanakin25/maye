# Mâye Pâtissier Dijital Menü

Bu klasör GitHub Pages için hazırdır.

## Yayınlama

1. GitHub'da yeni bir repository oluşturun.
2. `index.html` ve `styles.css` dosyalarını repository ana dizinine yükleyin.
3. **Settings → Pages** bölümüne gidin.
4. **Deploy from a branch** seçeneğini seçin.
5. Branch olarak `main`, klasör olarak `/ (root)` seçin.
6. Kaydedin. Birkaç dakika sonra sayfa adresiniz oluşur.
7. Oluşan adresi bir QR kod üreticisinde kullanın.

## Fiyat güncelleme

`index.html` içindeki ilgili `<strong>...</strong>` alanını değiştirmeniz yeterlidir.
